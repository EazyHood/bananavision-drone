# References

These references guided the architecture. They are not bundled datasets or model weights.

1. Neupane, Horanont, Hung. "Deep learning based banana plant detection and counting using high-resolution RGB images collected from UAV." PLOS One, 2019. Data availability points to Figshare DOI `10.6084/m9.figshare.7981547`. As of this repo creation, the Figshare article metadata is public and CC BY 4.0, but its public files API returns no downloadable files.
   https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0223906

2. Johansen et al. "Detection of Banana Plants Using Multi-Temporal Multispectral UAV Imagery." Remote Sensing, 2021.
   https://www.mdpi.com/2072-4292/13/11/2123

3. He et al. "Iterative optimization annotation pipeline and ALSS-YOLO-Seg for efficient banana plantation segmentation in UAV imagery." Frontiers in Plant Science, 2025.
   https://www.frontiersin.org/journals/plant-science/articles/10.3389/fpls.2024.1508549/full

4. Ultralytics YOLO segmentation and export documentation.
   https://docs.ultralytics.com/tasks/segment/
   https://docs.ultralytics.com/modes/export/

5. Roboflow Universe includes banana/tree drone datasets and hosted model examples. These can be useful for exploration, but should not be treated as proof of field accuracy without checking license, export format, class definitions, and validation split quality.
   https://universe.roboflow.com/count-banana-plants/count-banana-plants
   https://universe.roboflow.com/drone-tree-species-identification/tree-species-tr73h
