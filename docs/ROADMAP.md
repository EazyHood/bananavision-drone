# Roadmap

## v0.1

- Baseline RGB detector.
- YOLO segmentation wrapper.
- Geospatial outputs.
- CLI/API.

## v0.2

- Tile stitching for very large orthomosaics.
- Cross-frame duplicate suppression for in-flight video.
- Active-learning queue for uncertain detections.
- Web UI for reviewing low-confidence clusters.

## v0.3

- SAM/SAM2-assisted annotation workflow.
- Multi-spectral feature stack.
- Signed model manifests and remote artifact storage.
- Jetson benchmark suite.

## v1.0

- Locked validation protocol.
- Reference trained model for a public banana-UAV dataset.
- Field report template.
- Stable API contract.
